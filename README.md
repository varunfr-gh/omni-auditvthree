# OmniAudit

Real-time AI Compliance & Oversight Platform.

## What this does
OmniAudit intercepts AI agent actions before execution, evaluates them against deterministic compliance rules, and logs human-readable audit events with optional AI explanations.

## Core guarantees
- Deterministic enforcement
- Privacy-safe logging
- Real-time kill-switch
- Human-readable compliance logs

Built for hackathon demo purposes.
